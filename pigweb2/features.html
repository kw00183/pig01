Game Features

Select First Player at Random

Feature allows players the ability to randomly choose which player (human or computer) starts the game.

Turn Roll Log

Feature allows the user to see a log of all dice pairs rolled in the turn. This feature is especially nice when the user implements the random or greedy strategies for the computer or continues to roll as the human player, because you can see each roll logged in the turn.
