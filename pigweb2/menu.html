Menu Options

Game :: Exit

Exit allows the players to exit the game at any time during play. Please select Game > Exit or use CTRL-X as a shortcut to complete the operation.

Game :: New Game

New Game allows the players to start a new game at any time during play. Please select Game > New Game or use CTRL-N as a shortcut to complete the operation.

Strategy :: Cautious

Cautious strategy allows the computer player to roll one time per turn, hence the name. Please select Strategy > Cautious in the menu, or use CTRL-C as a shortcut to complete the operation.

Strategy :: Random

This strategy allows the computer player the option to let statistics decide the fate of the next roll in the turn. However, the random option is only allowed to randomly choose to roll additional times if the last role was valid (the dice pairs do not include a 1) and the game has not been won. Please select Strategy > Random or use CTRL-R as a shortcut to complete the operation.

Strategy :: Greedy

The greedy strategy gives the computer player the option of rolling up to three times in the turn, provided each of the rolls are valid (the dice pairs do not include a 1) and the game has not been won. Please select Strategy > Greedy or use CTRL-E as a shortcut to complete the operation.
